Proyecto Cards Seekers: tienda online TCG (Pokémon, Yu-Gi-Oh!, One Piece). React 18 + TypeScript + Vite + Chakra UI v2 + React Router v6. Maquetado completo finalizado en marzo 2026.

**Why:** Tienda online de venta de productos TCG sellados y cartas sueltas.

**How to apply:** El backend y la integración real de MercadoPago están pendientes. El checkout es un mock — ver comentarios en CheckoutPage.tsx. Los datos de productos son estáticos en src/data/products.ts (sustituir por API calls al conectar backend).
